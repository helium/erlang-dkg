erlang-dkg
=====
[![Build Status](https://badge.buildkite.com/6933d2231530c47800b727a796af13ac90faabcffbe12af487.svg?branch=master)](https://buildkite.com/helium/erlang-dkg)

Build
-----

    $ make

Test
-----

    $ make test
